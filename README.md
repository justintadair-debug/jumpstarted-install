# JUMPSTARTED

**Your AI team, running 24/7.**

JUMPSTARTED turns your Mac into a 24/7 AI operations center. Your agents work around the clock — handling tasks, coordinating with each other, and reporting back to you via Discord.

## Install

One command:

```bash
curl -fsSL https://raw.githubusercontent.com/justintadair-debug/jumpstarted-install/main/bootstrap.sh | bash
```

Then run `jmp setup` to connect to Discord and activate your license.

## Requirements

- macOS 12+ (Apple Silicon recommended)
- 8GB RAM minimum
- Active internet connection

## Features

- **AI Agents**: Deploy multiple AI agents that work as a team
- **Discord Integration**: One-click setup — no bot tokens or developer portal needed
- **24/7 Operation**: Runs in the background on your Mac
- **Secure**: Everything runs locally on your machine

## Commands

```bash
jmp setup     # Run the setup wizard
jmp start     # Start JUMPSTARTED
jmp stop      # Stop JUMPSTARTED
jmp status    # Check license and system status
jmp logs      # View live logs
```
